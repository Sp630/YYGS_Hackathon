OUR OBJECTIVE

Welcome to YYGO!

We created this website because, as YYGS students, we found it extremely annoying to have to navigate between google maps and the boundary map constantly.

Knowing that you can risk exclusion from the program if you go beyond the boundaries, being able to avoid them seemed important to us.

In the map page, you'll be able to access a map with your position within the boundaries.
